A suite of computer vision utilities
====================================

About
-----

A selection of useful utilities for general computer vision tasks written in Python. 

Dependencies
------------

* OpenCV 2.3.1 or greater with Python bindings
* Numpy
* Scipy
* Matplotlib
* FFMpeg
